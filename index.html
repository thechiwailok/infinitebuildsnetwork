How to Join

Java Edition

  1. Open Minecraft Java Edition
  2. Go to Multiplayer
  3. Click "Add Server"
  4. Enter "Infinite Builds" as the server name
  5. Enter "chiwailok.minekeep.gg" as the server address
  6. Click "Done" and then "Join Server"

Bedrock Edition

  1. Open Minecraft Bedrock Edition
  2. Go to "Servers" tab
  3. Click "Add Server"
  4. Enter "Infinite Builds" as the server name
  5. Enter "chiwailok.minekeep.gg" as the server address
  6. Use port 19132
  7. Click "Save" and then "Join"
